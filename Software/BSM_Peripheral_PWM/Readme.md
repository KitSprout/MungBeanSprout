MungBeanSprout PWM Example
========
* Author  : [Hom](http://about.me/Hom)
* Update  : 2015/02/22

Description
========
MungBeanSprout PWM 的範例程式，不斷改變 PWM 的 Duty Cycle。

Hardware
========
* PWM  
PA6  - TIM3 CH4 - PWM3  
PA7  - TIM1 CH2 - PWM4  
PA9  - TIM3 CH1 - PWM1  
PA10 - TIM3 CH2 - PWM2  
PB1  - TIM1 CH3 - PWM5
